Workflow FAQ

Can Mesuro help with laser engraving calibration?

Yes. Use point coordinates or calibrated measurements to compare engraved test marks, inspect offsets, and document setup corrections from output images.

Do I need a physical scale reference?

Not always. Pixel coordinate mode is useful when you need relative positions or repeated point checks; use calibrated scale when you need real-world units.

Can I export the calibration result?

Yes. You can export an annotated image, PDF report, CSV data, or a reusable project file.
