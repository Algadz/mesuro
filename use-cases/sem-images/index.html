Microscopy image measurement

Measure microscopy images online

Use Mesuro when you need a fast calibrated workflow for microscopy images, including SEM exports: batch load an image set, apply one scale across images that share the same magnification, measure features, then export clean annotated results.

Practical workflow

1

Load one image or a batch

Open a single microscopy image or multiple same-scale exports together. Files stay local.

2

Calibrate once

Draw over a visible scale bar, enter its value, then apply that scale to the whole batch when magnification is shared.

3

Measure features

Measure distances, diameters, areas or coordinates with calibrated units.

4

Export for review

Download annotated images or measurement data for reports and discussion.

Workflow FAQ

Can Mesuro read scale bars automatically?

You set the calibration manually from the visible scale bar. That keeps the workflow simple and transparent.

Can I apply one scale to multiple images?

Yes. If several loaded images share the same scale, you can calibrate once and apply that scale across the batch.

Does this work for SEM images too?

Yes. SEM exports with a visible scale bar fit the same calibrated microscopy workflow.

Can I export annotated images and data?

Yes. You can export annotated images and CSV data from the browser workflow.
