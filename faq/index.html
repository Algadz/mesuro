Technical FAQ

Answers for calibration, privacy, PDFs and workflow use

Mesuro stays intentionally small, so the FAQ is where the operational detail lives.

Calibration Privacy Pricing Technical Workflow

Calibration

How do I calibrate scale?

Use the Scale tool, draw a line over an object with a known size, then enter the real-world value and unit. Mesuro applies that conversion to later measurements on the same page.

Can I apply one scale to multiple images?

Yes. If several loaded images share the same scale, you can calibrate once and apply that scale across the batch.

How accurate are measurements?

Accuracy depends on the reliability of your reference, the source resolution, and how precisely points are placed. For best results, use the longest dependable reference line available and zoom in when placing points.

How does perspective correction work?

Select the four corners of a known rectangle in the image. Mesuro applies a homographic correction so the photographed plane is flattened before measurement.

Privacy

Are files uploaded?

No. Images, PDFs, annotations and measurements stay on your device while you work.

Is processing local-only?

Yes. The measurement workflow runs in the browser. Mesuro only stores a short-lived device fingerprint hash for free-tier limit enforcement; the actual files never leave your device.

Does Mesuro work offline?

Once the app page has loaded, the measurement workflow can continue locally. File processing itself does not require an upload round trip.

Pricing

Is Mesuro free?

Yes. Mesuro is free for up to 20 measurements each day. Mesuro Pro removes the daily cap for frequent use.

What happens when I reach the free daily limit?

You can wait for the next daily reset or upgrade to Mesuro Pro for unlimited measurements.

Technical

What formats are supported?

Mesuro supports common image formats accepted by modern browsers and PDF files. It can also save and reload Mesuro project files for later continuation.

Can I measure PDF files?

Yes. You can open PDF pages directly, load page ranges or entire multi-page documents, and export annotated PDF reports or CSV data.

What can I export?

Annotated images, PDF reports, CSV data and project files. Exports are designed for practical handoff rather than presentation-heavy reporting.

Which browsers work best?

Use a current desktop or mobile browser with Canvas support. The app is optimized around modern browser capabilities rather than a packaged install.

Workflow

Can I use Mesuro for SEM image measurement?

Yes. Calibrate from the visible scale bar, measure structures on the image, and export the annotated result for technical review or reporting.

Can I use it for microscopy annotation?

Yes. The same calibrated workflow works for microscopy images when you have a reliable scale reference.

Can I measure technical blueprints?

Yes. Open the PDF or image, calibrate from a known dimension, then measure lengths, areas, angles or coordinates directly.

Can I use Mesuro for CAD drawing review?

Yes. Open an exported CAD drawing or technical drawing, calibrate from a known dimension, then measure distances, angles, areas or diameters directly.

Can Mesuro help with laser engraving calibration?

Yes. Use point coordinates or calibrated measurements to compare engraved test marks, check offsets, and document setup corrections from output images.

Can I measure from a known object in a photo?

Yes. If a reference object with a known size is visible in the same plane as the item you want to measure, use it to calibrate the photo before measuring.

Need the workspace rather than the explanation?

Open the app and start from a local image or PDF.
